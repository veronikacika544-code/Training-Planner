# Weather Diary / Дневник погоды

## Автор
[Кожешвиль Вероника]

## Описание программы
Weather Diary - это GUI-приложение для ведения дневника погоды. Программа позволяет:
- Добавлять записи о погоде (дата, температура, описание, осадки)
- Просматривать все записи в табличном виде
- Фильтровать записи по дате и температуре
- Сохранять данные в JSON-файл
- Загружать данные из JSON-файла

## Требования
- Python 3.6 или выше
- Tkinter (обычно входит в стандартную установку Python)
- Git (опционально, для работы с репозиторием)

## Как запустить приложение

1. Убедитесь, что у вас установлен Python 3.6 или выше
2. Скачайте или клонируйте репозиторий
3. Запустите программу:
   ```bash
   python weather_diary.py

# Random Quote Generator - Генератор случайных цитат

Автор: Вероника Кожешвиль

## Описание

Random Quote Generator - это десктопное приложение для генерации вдохновляющих и мотивирующих цитат. Программа содержит коллекцию предустановленных цитат, позволяет добавлять свои, сохраняет историю просмотра и поддерживает фильтрацию по авторам и темам.

### Основные возможности

- Генерация случайных цитат из коллекции
- Добавление пользовательских цитат с указанием автора и темы
- История всех сгенерированных цитат с датой и временем
- Фильтрация истории по автору и теме
- Сохранение коллекции цитат и истории в JSON
- Красивый и интуитивно понятный интерфейс

## Системные требования

- Python 3.7 или выше
- Стандартные библиотеки: tkinter, json, random, datetime

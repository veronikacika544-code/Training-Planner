# Weather Diary — Дневник погоды

**Автор:** Иван Иванов (пример)

## Описание
Простое GUI-приложение для ведения дневника погоды.  
Позволяет добавлять записи с датой, температурой, описанием и отметкой об осадках, фильтровать записи по дате и температуре, сохранять и загружать данные в формате JSON.

## Как запустить
1. Убедитесь, что установлен Python 3.6+.
2. Сохраните файл `weather_diary.py`.
3. Запустите командой:
   ```bash
   python weather_diary.py
